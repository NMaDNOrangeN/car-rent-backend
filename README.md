## Запуск

1. Создание venv (Один раз при установке проекта):\
`python -m venv .venv`

2. Активация venv\
`.venv\Scripts\activate`

3. Установка зависимости\
`pip install -r requirements.txt`

4. Запуск проекта:
    1. В режиме dev\
    `python -m fastapi dev main.py`
    2. В режиме prod\
    `python -m fastapi run main.py`
